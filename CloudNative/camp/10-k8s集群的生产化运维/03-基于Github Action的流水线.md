# 基于Github Action的流水线



### 基于公共GitHub的action构建流水线

* 低成本
  * GitHub目前为项目提供免费构建流水线，可满足日常构建需求

* 免运维
  * 无需自己构建流水线，GitHub提供小量构建请求

* 易构建
  * GitHub action非常容易构建，只需点击几次按钮，即可完成
  * GitHub提供了一系列内建action, 社区有大量可复用的action
* 易集成
  * 无需配置GitHub Webhook即可完成与PR的联动
