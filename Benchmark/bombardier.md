# bombardier

Fast cross-platform HTTP benchmarking tool written in Go

github




去 release 页面下载对应二进制文件即可。



使用

```shell
bombardier -c 125 -n 10000000 http://localhost:8080
# -c 线程数
# -n 总次数
```
